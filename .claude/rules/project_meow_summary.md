# PRD: 「야옹아, 내가 집사라도 괜찮을까?」 인터랙티브 랜딩 페이지

## 프로젝트 개요

### 목적
만화 「야옹아, 내가 집사라도 괜찮을까?」 도서 홍보를 위한 체험형 인터랙티브 웹사이트.
사용자가 주인공 '미정'이 되어 고양이와 가까워지는 서사를 직접 경험하며, 책과 고양이에 대한 관심을 자연스럽게 유도한다.

### 핵심 컨셉
- 실제 만화 컷(디지털 작업물)을 활용한 웹 버전 스토리 구성
- 흑백 만화 → 파스텔 온기로 변화하는 컬러 연출
- 종이 질감 + 패럴랙스 스크롤로 만화책을 넘기는 감각 재현
- 챕터별 인터랙티브 맵/미니게임으로 능동적 참여 유도

### 타겟 사용자
| 구분 | 대상 |
|------|------|
| 주 타겟 | 20-30대 1인 가구 여성, 인생의 전환기를 겪고 있는 사람, 고양이 입양을 망설이는 예비 집사 |
| 부 타겟 | 인터랙티브 콘텐츠에 관심 있는 MZ세대, 반려동물 입양 고민 중인 모든 연령대 |
| 핵심 심리 | "나 같은 사람도 괜찮을까?"라는 자기 의심 |

### 레퍼런스
- **The Boat (SBS)**: 스크롤 기반 인터랙티브 스토리텔링
- **Machinarium**: 포인트 앤 클릭 공간 탐색
- **Concept Capers**: 카드 인터랙션 (챕터2 타로카드)

## 챕터 1: 처음 만나는 고양이

| 항목 | 내용 |
|------|------|
| 서브타이틀 | CHAPTER 1 |
| 타이틀 | 처음 만나는 고양이 |
| 공간 | 미정의 방 → 공원 |
| 개요 | 이혼 후 거처를 옮긴 미정이 공원에서 고양이 탐정 소녀 '미래'를 만나, 난생처음 고양이에게 다가가는 법을 배우며 작은 설렘을 느낀다. |

### 디스크립션
- 이혼 후 새로운 곳으로 이사를 간 미정은 한참 이삿짐 정리 중이다.
- 이리저리 고양이를 찾는 미정의 곁에 초등학생으로 보이는 어린이가 다가와 말을 걸었다.

### 인터랙티브 맵 - 공원

| 항목 | 내용 |
|------|------|
| 목표 | 공원 곳곳을 클릭해 고양이 찾기 |
| 커서 | 참치캔 모양 |
| 클릭 영역 | 벤치 아래, 풀숲, 나무 뒤 등 |
| 완료 조건 | 모든 고양이 발견 시 다음 진행 |

### 인터랙티브 게임 - 꽃님이와 코인사

| 항목 | 내용 |
|------|------|
| 상황 | 미래가 꽃님이를 소개하며 인사해보라고 권함 |
| 디스크립션 | "천천히 손가락을 꽃님이의 코에 가져다 주세요." |
| 조작 | 커서(손가락)를 천천히 꽃님이 코 쪽으로 이동 |
| 실패 | 너무 빠르면 꽃님이가 움츠러듦 |
| 성공 | 천천히 다가가면 킁킁 → 핥음 |
| 보상 연출 | 화면 전체에 따뜻한 분홍색 온기 퍼짐 |

## 챕터 5: 나란히 앉아오는 봄

| 항목 | 내용 |
|------|------|
| 서브타이틀 | CHAPTER 5 |
| 타이틀 | 나란히 앉아오는 봄 |
| 공간 | 미정의 방 → 공원(피크닉) → 귀갓길 |
| 개요 | 사건을 겪으며 성장한 미정이 동물보호 활동 및 새로운 행보를 결심하고, 봄날 벚꽃 아래에서 모로, 미래와 함께 나란히 앉아 따뜻한 일상을 맞이한다. |

### 디스크립션
- 사건 이후, 미정은 진정한 홀로서기의 한 걸음을 내디뎌 본다.
- 미정은 잠시 뜸들이다 입을 열었다.

### 스토리 흐름
1. 미정의 방 — 죽은 줄 알았던 선인장이 싹을 틔움 (성장/새 시작 상징)
2. 겨울이 지나 봄 — 모로, 미래와 공원에서 만남, 사건 회포, 새 계획 공유
3. 피크닉 — 셋이 돗자리에 앉아 꽃잎과 눈 즐김
4. 시간 경과, 또다시 봄 — 미정이 새 여정을 마치고 귀갓길에 '겨울이'와 재회

### 연출
| 요소 | 설명 |
|------|------|
| 컬러 전환 | 흑백 → 파스텔톤으로 점진적 채색 |
| 선인장 | 스크롤에 따라 흑백 → 연두색 채워짐 |
| 파티클 | 화면에 흩날리는 꽃잎/눈 효과 |

### 인터랙티브 맵 - 피크닉

| 항목 | 내용 |
|------|------|
| 구성 | 세 사람(미정, 모로, 미래)이 돗자리에 앉아 하늘을 바라보는 장면 |
| 컬러 | 스크롤 도달 시 따뜻한 파스텔톤으로 변환 |
| 시차 효과 | 마우스 커서 따라 공간이 입체적으로 움직임 |
| 파티클 | 드래그 시 꽃잎/눈송이가 커서 방향으로 휘날림 |

### 쿠키 챕터

| 단계 | 연출 |
|------|------|
| 1 | 화면 암전 후 밝아짐 |
| 2 | 쿠키 만화 등장 — 여행에서 돌아온 미정이 겨울이를 만남 |

## 부록: 입양고사 체크리스트

| 항목 | 내용 |
|------|------|
| 목적 | 책의 마지막 콘텐츠를 인터랙티브하게 체험 |
| 인터랙션 | 문항 체크할 때마다 배경에 꽃 한 송이씩 피어남 |
| 완료 | "당신은 이미 꽤 괜찮은 집사입니다" + 책 구매 링크 |

### 체크리스트 항목
1. 고양이의 식비와 병원비, 필요 물품 등의 유지비(월평균 15만원)를 꾸준히 지출할 수 있는 경제력
2. 하나의 생명을 끝까지 보살필 수 있는 책임감
3. 고양이의 일거수일투족 모든 생활을 돌봐줄 수 있는 체력
4. 함께할 수 있는 시간과 마음의 여유
5. 나의 가족이 고양이 알레르기, 천식 등으로 인해 위험해지지 않는지
6. 주거 공간이 고양이를 키울 수 있는지
7. 가구와 생활 도구가 고양이의 마킹과 호기심으로 망가져도 이해할 수 있는 인내심과 배려
8. 중성화 수술에 대한 필요성 인식과 시행 계획 여부
9. 15년 이상을 함께할 수 있는지, 반려로서의 확고함

### CTA
| 항목 | 내용 |
|------|------|
| 문구 | "미정의 이야기 전체 보기" (비공격적) |
| 링크 | 책 구매 사이트 |
