# UX Flow: 「야옹아, 내가 집사라도 괜찮을까?」

## 문서 개요

이 문서는 인터랙티브 랜딩 페이지의 사용자 경험 흐름을 정의합니다.
기획, UX, 필요 컴포넌트 중심으로 정리했습니다.

> ⚠️ 현재 히어로, 챕터1, 챕터5만 구성 완료. 챕터 2,3,4는 추후 보충 예정.

---

## 전체 플로우

```
[히어로] → [CH1: 공원] → [CH2: 공방] → [CH3: 골목] → [CH4: 빌라] → [CH5: 피크닉] → [쿠키] → [부록] → [CTA]
             │              │            │            │             │           │         │
         고양이찾기      타로게임      밥주기       단서찾기      컬러전환    서프라이즈   입양고사
         +코인사        +공방탐색                                +3D시차
```

### 예상 체험 시간

| 구간 | 시간 |
|------|------|
| 히어로 | 10~15초 |
| 챕터 1 | 2~3분 |
| 챕터 2~4 | 각 2~3분 (예상) |
| 챕터 5 + 쿠키 | 2~3분 |
| 부록 | 1~2분 |
| **전체** | **약 15분** |

## 공통 UX 규칙

### 스크롤 진행
- 만화 컷들이 패럴랙스로 배치되며 스크롤에 따라 이야기 전개
- 특정 지점 도달 시 애니메이션/사운드 자동 재생
- 화면 한쪽에 현재 챕터 위치 표시

### 인터랙티브 맵 공통
| 규칙 | 설명 |
|------|------|
| 호버 | 클릭 가능 영역에 시각적 힌트 |
| 클릭 | 애니메이션 + 효과음 |
| 완료 조건 | 필수 요소 발견 시 다음 진행 |
| 힌트 | 30초 무반응 시 힌트 표시 |

### 미니게임 공통
| 규칙 | 설명 |
|------|------|
| 실패 | 재시도 가능, 페널티 없음 |
| 피드백 | 성공/실패 시 명확한 반응 |

### 컬러 톤 변화
| 구간 | 톤 |
|------|-----|
| 히어로 ~ 챕터 3 | 흑백 (mono) |
| 챕터 4 | 어두운 (dark) |
| 챕터 5 ~ 부록 | 파스텔 (pastel) |

## 챕터 1: 처음 만나는 고양이

### 정보
| 항목 | 내용 |
|------|------|
| 서브타이틀 | CHAPTER 1 |
| 타이틀 | 처음 만나는 고양이 |
| 공간 | 미정의 방 → 공원 |

### 목적
- 미정의 상황 소개 (이혼 후 새 출발)
- 첫 고양이와의 만남 체험
- 고양이 접근법 체험 (코 인사)

### 플로우
```
[챕터 타이틀 + 디스크립션]
    ↓
[이삿짐 정리 만화 컷] — 패럴랙스 스크롤
    ↓
[선인장 있는 방, 창밖 고양이]
    ↓
[참치캔 들고 나가는 장면]
    ↓
[인터랙티브 맵: 공원] — 고양이 찾기
    ↓
[미래 등장]
    ↓
[인터랙티브 게임: 코인사]
    ↓
[성공 → 분홍빛 온기 연출]
    ↓
[챕터 2로]
```

### 인터랙티브 맵 - 공원

**화면 구성**
```
┌─────────────────────────────┐
│         [나무]              │
│                             │
│  [풀숲]            [벤치]   │
│                             │
│       [산책로]              │
│                             │
│  ◎ 커서(참치캔)             │
└─────────────────────────────┘
```

**클릭 영역**
| 영역 | 반응 | 결과 |
|------|------|------|
| 벤치 아래 | 눈 반짝 | 고양이 발견 |
| 풀숲 | 풀 흔들림 | 고양이 발견 |
| 나무 뒤 | 꼬리 보임 | 고양이 발견 |

**완료 조건**: 모든 고양이 발견 → 미래 등장

### 인터랙티브 게임 - 꽃님이와 코인사

**화면 구성**
```
┌─────────────────────────────┐
│                             │
│      ┌─────────────┐        │
│      │   꽃님이    │        │
│      │    🐱      │        │
│      │    ↑코     │        │
│      └─────────────┘        │
│                             │
│        ◎ 커서(손가락)       │
│                             │
│  [힌트 텍스트 영역]         │
└─────────────────────────────┘
```

**상태별 반응**
| 상태 | 조건 | 꽃님이 반응 | 힌트 |
|------|------|------------|------|
| 대기 | 커서가 멀리 | 경계하며 바라봄 | "천천히 손가락을 가져다 주세요" |
| 접근 중 | 천천히 다가감 | 귀가 앞으로 | "좋아요, 천천히..." |
| 너무 빠름 | 급하게 움직임 | 움츠러듦 | "너무 빨라요!" |
| 성공 | 코 근처 3초 유지 | 킁킁 → 핥음 | (연출로 전환) |

**성공 연출**
1. 꽃님이가 손가락 핥음
2. 화면에 분홍빛 온기 퍼짐
3. 다음 장면으로 전환

### 필요 컴포넌트
| 컴포넌트 | 역할 |
|----------|------|
| `ChapterTitle` | 챕터 타이틀 + 디스크립션 |
| `ParkMap` | 공원 인터랙티브 맵 |
| `ClickableArea` | 클릭 가능 영역 |
| `NoseKiss` | 코인사 미니게임 |
| `CustomCursor` | 참치캔/손가락 커서 |

### 플로우
```
[챕터 타이틀 + 디스크립션]
    ↓

    ↓
[공원에서 모로, 미래와 만남]
    ↓
[대화 컷들]
    ↓
[인터랙티브 맵: 피크닉] ← 3D 시차 + 파티클

[시간 경과]
    ↓
[쿠키: 암전 → 겨울이 재회]
    ↓
[부록으로]
```

### 연출 상세

**선인장 컬러 전환**
```
[스크롤 시작]
    ↓
[선인장 흑백]
    ↓ 스크롤 진행
[점차 채색] → 새싹: 연두색
    ↓
[완전히 컬러풀]
```

### 인터랙티브 맵 - 피크닉

**화면 구성**
```
┌─────────────────────────────┐
│           [하늘]            │
│      ☁️         ☁️          │
│                             │
│   [미정] [모로] [미래]      │
│        [돗자리]             │
│                             │
│  [풀밭]              [나무] │
│                             │
│    🌸 흩날리는 꽃잎/눈 ❄️   │
└─────────────────────────────┘
```

**인터랙션**
| 동작 | 효과 |
|------|------|
| 스크롤 도달 | 흑백 → 파스텔톤 전환 |

| 드래그 | 꽃잎/눈송이가 커서 방향으로 휘날림 |

## 부록: 입양고사 체크리스트

### 목적
- 책의 핵심 콘텐츠 체험
- 자기 점검 기회
- 자연스러운 구매 유도

### 플로우
```
[인트로 텍스트]
    ↓
[문항 1] → 체크 → 꽃 1송이
    ↓
[문항 2] → 체크 → 꽃 2송이
    ↓
... (9개 문항)
    ↓
[모두 체크]
    ↓
[꽃밭 + "당신은 이미 꽤 괜찮은 집사입니다"]
    ↓
[CTA 버튼: "미정의 이야기 전체 보기"]
```

### 화면 구성
```
┌─────────────────────────────┐
│                             │
│  입양고사                    │
│                             │
│  ☐ 문항 1                   │
│  ☐ 문항 2                   │
│  ☐ 문항 3                   │
│  ...                        │
│                             │
│  🌸 🌸 🌸 (체크마다 추가)    │
│                             │
│  [ 미정의 이야기 전체 보기 ] │
└─────────────────────────────┘
```

### 체크 인터랙션
| 상태 | 반응 |
|------|------|
| 미체크 | 빈 박스 |
| 호버 | 살짝 확대 |
| 체크 | 체크 표시 + 바운스 + 꽃 추가 |
| 모두 완료 | 꽃잎 흩날림 → 완료 메시지 |
